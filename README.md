### 📃Description

- 금융 전문 커뮤니티 애플리케이션 제작

### 🎯GOAL

- 금융상품 데이터 기반 예금 및 적금 상품 소개
- 금융 상품 추천 알고리즘 구성
- 환율 정보 API를 활용한 환율 계산 서비스 구성
- 지도 API를 활용한 은행 검색 서비스 구성
- 커뮤니티 서비스 구성
- 서비스 관리 및 유지보수 목표
